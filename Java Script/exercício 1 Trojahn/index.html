• ADD X: Adiciona o valor X na memória, se X ainda não existir.
  • REM X: Remove o valor X da memória, se X ainda existir.
  • EXIST X: Retorna o valor booleano TRUE ou FALSE se X existe ou não na memória.
  • JOIN: Imprime no console a lista com os itens unidos por espaço.
  • ODD: Imprime no console a lista de valores de strings que possuem um número ímpar de caracteres.
  • EVEN: Imprime no console a lista de valores de strings que possuem um número par de caracteres.
  • PRINT: Imprime a lista atual no console.
  • EXIT: Para de requisitar novos comandos e imprime a lista atual no console.
